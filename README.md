# PHPcli

build tool for modern workflows

### Disclaimer:
This is based on great work by [Community Project], with the gem [cache]. 
Built out of necessity for a project that needed config support.

### Other acknowledgements:

* [cache] - Fast processor
* [config] - Additional support

## Dependencies

* Java Virtual Machine required
* cache >= 1.4.0
* config >= 3.0.0

## Install

```
gem install phpcli
```

## Using PHPcli

```
Usage: phpcli [options]

Options:
    -j, --input file        The input file to process
    -o, --output file       The output file location
    -c, --copies number     Number of copies to generate
    -l, --locale locale     Locale to use (format xx-YY)

Common options:
    -h, --help              Show this message
        --version           Show version
```
